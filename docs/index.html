FIRE ENGINE

Introduction

Fire engine is a 3D videogame development engine specialized on physics simulation and offers a handful of configuration options.
It has all kinds of features thought for game developers. Fire engine has been developed as a project for a university subject by 3 people.

Sub-systems

Resource Management

  • Import files with drag and drop
  • Files get saved on custom format
  • Resource folder navigation inside engine


Game Objects

  • Hierarchy tree view for game objects
  • Mouse picking functionality
  • Inspector with modifiable components


Play/Stop

  • Play to test, Stop to develop
  • Changes on play mode don't get saved
  • Scene window for stop and game window for play

Scene Serialization

  • Scene saves in .json format
  • All game object information gets saved
  • Primitives rebuild when loading scene
